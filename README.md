# 顯微測量校正

一個給國中／高中生使用的互動式顯微測量模擬器，主題是 **目鏡測微器 × 載物台測微器** 的校正與換算。

目前版本：**v1.1 課堂測試版**

## 專案目的

學生在學習顯微測量時，常見困難包括：

- 分不清楚「目鏡測微器」與「載物台測微器」的位置、功能與差異。
- 不理解為什麼不同倍率下，目鏡測微器每格代表的實際長度會改變。
- 不熟悉用 `x = 10b / a` 進行校正與換算。
- 高倍物鏡下誤用粗調節輪，容易撞擊載物台測微器玻片。

本模擬器透過顯微鏡操作、視野刻度疊合、安全距離提示與待測標本測量，協助學生建立「先校正，再測量」的概念。

## 教學重點

### 目鏡測微器

- 位在目鏡內。
- 在視野中始終清楚可見。
- 刻度固定為 100 格。
- 可旋轉角度，但線段中心固定在視野中央。

### 載物台測微器

- 是放在載物台上的標準玻片。
- 每 1 小格代表 `10 μm`，也就是 `0.01 mm`。
- 會受對焦影響，未對焦時會模糊或看不清楚。
- 用來校正目鏡測微器每格的實際長度。

### 校正結果

模擬器採用課堂常見的校正關係：

| 總倍率 | 對齊關係 | 每格目鏡刻度長度 |
|---|---:|---:|
| 40X | 40 格 a 對上 100 格 b | 25 μm |
| 100X | 100 格 a 對上 100 格 b | 10 μm |
| 400X | 100 格 a 對上 25 格 b | 2.5 μm |

## 操作流程

1. 點選顯微鏡圖上的 `4X` 低倍物鏡開始。
2. 拖曳目鏡，旋轉目鏡測微器，讓藍色刻度與橘色刻度平行。
3. 拖曳粗調節輪，讓載物台測微器進入焦距範圍。
4. 拖曳細調節輪調清楚。
5. 移動玻片位置，讓第一格線對齊。
6. 記錄該倍率校正結果。
7. 點選物鏡切換至 `100X`、`400X`，高倍只用細調節輪微調。
8. 完成三倍率校正後，測量未知標本。

## 待測標本

完成校正後，可切換並測量三種標本：

| 標本 | 最長端長度 |
|---|---:|
| 草履蟲 | 150 μm |
| 植物表皮細胞 | 40 μm |
| 白血球 | 20 μm |

## 高倍安全設計

在 `400X` 高倍物鏡下，模擬器會提醒學生：

- 高倍物鏡工作距離短。
- 不應使用粗調節輪大幅移動載物台。
- 若物鏡接近玻片，會出現危險距離提示。
- 若繼續操作，玻片會破裂。

## 使用方式

### 直接開啟

下載專案後，直接用瀏覽器開啟：

```text
outputs/index.html
```

或開啟同內容檔案：

```text
outputs/microscope_measurement_calibration_simulator.html
```

### GitHub Pages

如果要放到 GitHub Pages，可將 Pages 的發布來源設定到：

```text
outputs/
```

首頁檔案已提供為：

```text
index.html
```

## 建議操作環境

- 電腦瀏覽器
- 平板橫向

手機直向畫面較窄，仍可瀏覽，但不建議作為主要課堂操作方式。
